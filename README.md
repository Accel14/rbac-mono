# Structure
```bash
├── frontend/            # Angular
├── backend/             # NestJS
├── docker-compose.yaml
├── README.md
```

# Docker
```bash
docker-compose up --build
```
Access at: [localhost:4200](http://localhost:4200)
# DOCS (Swagger)
At [localhost:3000/api/docs](http://localhost:3000/api/docs)
# Local deployment
```bash
cd backend
npm run start:dev
cd ../frontend
ng serve
```
Access at: [localhost:4200](http://localhost:4200)
